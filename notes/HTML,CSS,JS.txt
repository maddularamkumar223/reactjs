Arrow function

It is known as syntactic sugar.

Normal function Syntax

function functionName([parametr]){//code}

Arrow Function Syntax

variable declaration identifier = ([p]) => {//code}
identifier([a])

1. If I have only one line of execution

	let demo = () => clg("I am a demo function")

2. If I don't have any parameters no need to use the parentheses apart from that we can use "$/_"

	let demo1 = $ => clg('I am a demo1 function')

3. If I have a only one parameter no need to pass the parentheses
	let displayName = name => clg(name)
